Descripción del Proyecto: Sistema de Gestión de Tokens JWT

Este proyecto presenta una solución básica para la generación y utilización de tokens JWT en llamadas a API. Ha sido desarrollado utilizando C# y ASP.NET en el entorno de desarrollo Visual Studio.

El sistema permite a los usuarios autenticarse y obtener un token de acceso. Este token, una vez obtenido correctamente, puede ser utilizado para realizar llamadas a API en los diferentes módulos del sistema.

En resumen, el proyecto ofrece una implementación de backend para la gestión de tokens JWT, facilitando la autenticación y autorización en las llamadas a la API.
